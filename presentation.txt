Présentation : 

Happydays est un site permettant aux utilisateurs connectés de pouvoir écouter de la musique (uploadée au préalable par l'utilisateur) 
via un player intégré sur le site.

Le site comprend plusieurs pages ainsi qu'une base de donnée. 

l- la structuration du site : 
    // Connexion
    - une page d'inscription pour l'utilisateur (inscription.php)
    - une page de connexion pour l'utilisateur (index.php)

    // Player
    - une page principale du site regroupant le player et les playlists (dashboard.php)
    
    // Fonctionnalité

    - une page d'affichage des playlists et  tailles sous forme de table (playlist_table.php)
        - une page formulaire pour ajouter/modifier des playlists (form_playlist.php)


    - une page d'affichage des musiques et appartenances sous forme de table (music_table.php)
        - une page formulaire pour ajouter/modifier des musiques (form_music.php)

    // Contact

    - une page de la charte à accepter lors de l'inscription (charte.php)
    - une page de renseignement du site (about.php)
    - une page de contact comprenant un formulaire ect.. (contact.php)

2- Base de donnée :

bdd: happy_days
